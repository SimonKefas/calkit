CalKit

A zero-dependency calendar suite built with Web Components. Date picker, booking calendar, time picker, and resource scheduler.

Custom Event Content

Use the eventContent property to render custom HTML inside event blocks. Set show-event-time="false" to hide the default time row.

Events display custom metadata
scheduler.eventContent = (event, resource) => { const el = document.createElement('div'); el.innerHTML = `<strong>${event.title}</strong>`; if (event.metadata?.client) { el.innerHTML += `<div style="font-size:10px;opacity:0.8">Client: ${event.metadata.client}</div>`; } return el; };

Drag to Move & Resize

Set draggable-events to enable drag-to-move and resize handles. Events with locked: true cannot be dragged (Room B/C events below). Drag on empty space to create. Use snap-interval for fine-grained snapping (15 min below on a 30 min grid). min-duration/max-duration constrain event length.

Drag an event to move it, or drag its bottom edge to resize (15min snap, 15min–3hr range)
<cal-scheduler view="week" draggable-events snap-interval="15" min-duration="15" max-duration="180"> </cal-scheduler> scheduler.addEventListener('cal:event-move', (e) => { const { event, from, to } = e.detail; // Update event in your data source }); scheduler.addEventListener('cal:event-resize', (e) => { const { event, from, to } = e.detail; // Update event endTime });
